# บทที่ 1 ประเภทของคำและลักษณะของคำนาม

# ประเภทของ “คำ” (الكَلِمَةُ)

คำในภาษาอาหรับแบ่งออกเป็น 3 ประเภท

  1. นาม (اِسمٌ)
  2. กริยา (فِعلٌ)
  3. ฮัรฟ์ (حَرفٌ)

# 4 ลักษณะของ “คำนาม”

# 1. การเปลี่ยนแปลงส่วนท้ายของคำ (إِعرَابٌ)

  1. رَفعٌ
  2. نَصبٌ
  3. جَرٌّ

# 2. จำนวน (عَدَدٌ)

  1. จำนวน 1 (مٌفرَدٌ)
  2. จำนวน 2 (مُثَنَّى)
  3. จำนวน 3+ (جَمعٌ)

# 3. เพศ (جِنسٌ)

  1. ชาย (مُذكَّرٌ)
  2. หญิง (مُؤَّنَثٌ)

# 4. เจาะจง / ไม่เจาะจง (قِسمٌ)

  1. เจาะจง​ (مَعرِفَةٌ)
  2. ไม่เจาะจง (نَكِرَةٌ)
